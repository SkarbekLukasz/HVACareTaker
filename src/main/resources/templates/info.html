Informacje o projekcie

Aplikacja HVACareTaker powstała w ramach samodoskonalenia w zakresie projektowania i programowania aplikacji opartych na obiektowym języku programowania Java oraz jego framework'u Spring,a takze innych twoarzyszących technologii.

Produkcyjnym celem aplikacji jest realizacja zadań obecnych w codziennej pracy techników i inżynierów odpowiedzialnych za utrzymanie w ruchu urządzeń wentylacyjnych i klimatyzacyjnych w obsługiwanych obiektach.

Merytorycznym celem aplikacji jest przyswojenie i zastosowanie w praktyce zdobytej wiedzy z zakresu projektowania apliakcji webowych, a także jej rozwój i rozbudowę o kolejne funkcjonalności/technologie obecne w pracy na stanowisku Software Developera. Jest też miejscem testów niewykorzystywanych wcześniej rozwiązań i możliwości, które daje pakiet zależności Spring Boot.

Technologie wykorzystywane w aplikacji:
  • Java 11
  • Spring Boot 2.6.4
  • Thymeleaf
  • Spring Security
  • Spring Data JPA
  • Spring Web
  • Spring DevTools
  • Spring Mail
  • Spring Validation
  • Maven
  • Git&GitHub
  • H2 Database
  • MySQL
  • Liquibase
  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • Font Awesome
